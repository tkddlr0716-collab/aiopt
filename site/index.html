배포 전에, 비용사고를 먼저 잡자.
baseline 사용 로그를 기반으로 변경(모델/컨텍스트/리트라이/트래픽)을 시뮬레이션하고, 경고 텍스트 + Exit code 0/2/3로 반환합니다.
5분 체험 Pro 구매 npm Docs

Quickstart

$ npx aiopt install --force
$ npx aiopt guard --context-mult 1.2 --call-mult 10
$ echo $?   # 0 OK / 2 WARN / 3 FAIL
저장 파일: aiopt-output/guard-last.txt, aiopt-output/guard-last.json

CI/PR에서 보이게

  • Step Summary에 자동 출력
  • WARN/FAIL이면 PR 코멘트 자동 작성
  • exit=3을 머지 차단으로 사용

로컬 대시보드

  • npx aiopt dashboard --port 3010
  • 127.0.0.1 바인딩(로컬 전용)

결제(준비 중)

  • Polar Checkout
  • 오프라인 라이선스 키: aiopt license activate
Checkout 링크는 추후 Polar 상품 URL 확정 후 연결됩니다.
